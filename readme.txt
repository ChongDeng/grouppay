App Major Features:

1.User Related
	a.Registration
	b.Login and logout

2.Group Related
	a.Maintain an owner and a treasurer
	b.Find and request to join a group by group name search
	c.List all group that current user has enrolled and click to see the group detail
	d.Group member can create event for the group
	e.Owner can invite user to join the group by username search or friend

3.Finance Related
	a.Prepay credit to the personal account
	b.Transfer credit to one of the group enrolled in
	c.Pay credit to the event attend
	d.Periodically pay to one group

4.Friend Related
	a.Add friend by username search
	b.List all the friend of the current user

5.Event Related
	a.Maintain the total expense and how the expense paid
	b.Expense report by user and evidence upload
